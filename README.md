# <span style="color: #00BE67;">GooseOS</span>

A chaotic Win95-like desktop playground complete with: a game where a goose runs from affine matrices, users upload and vote on goose art, and everything boots through a fake `blue screen of death`... but then the __eldritch software corruption__ begins.

> __NOTE!__: *throughout this repo you may see the word __gooses__. I'm fully aware that it's TECHNICALLY spelled __geese__. But __gooses__ is hilarious; leave me alone; this is a safe space for __gooses__, __goosen__, and __goosing__.*
